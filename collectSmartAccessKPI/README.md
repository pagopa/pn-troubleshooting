# compare configurations files between environment

## Execution

`node generateCvs <awsProfile>`

Where:
- awsProfile: profile of the AWS account where the script will be executed

## Example

`node generateCvs sso_pn-core-pro`

## Produces

`Report_KPI_LogInsight-<startDate>, <endDate>.xlsx`

where:

- startDate: start date of the period for which the report is generated
- endDate: end date of the period for which the report is generated
